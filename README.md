# 🎨 ITD Imager

> Веб-инструмент для подготовки и автозагрузки изображений (баннеров/постов) на платформу **ИТД**.

![HTML](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

## ✨ Возможности

- **Загрузка изображений** — drag & drop или выбор файла (PNG, JPG, WebP, GIF)
- **Обрезка и масштабирование** — интерактивный редактор с зумом, панорамированием и сеткой третей
- **Два режима работы:**
  - ⚡ **API** — генерирует скрипт для прямой загрузки через API платформы
  - 🎨 **Рисование** — квантизация цветов (Median-Cut) с предпросмотром палитры
- **Настройки** — выбор размера холста (Баннер 1100×380 / Пост 800×500), формата (PNG, JPEG, WebP) и качества
- **📱 Мобильная версия** — инжектор `m.js` для загрузки баннеров прямо с телефона
- **Адаптивный дизайн** — тёмная тема, современный UI

## 📦 Установка

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/nersuga/itd_imager.git
   cd itd_imager
   ```

2. **Зависимости не требуются** — проект работает на чистом HTML/CSS/JS без сборщиков.

## 🚀 Запуск

Откройте файл `index.html` в браузере — или запустите через любой локальный сервер:

```bash
# Python
python -m http.server 8080

# Node.js (npx)
npx serve .

# VS Code — расширение Live Server
```

Затем перейдите по адресу `http://localhost:8080`.

## 🛠 Как пользоваться (ПК)

1. Загрузите изображение (перетащите или нажмите на область загрузки)
2. Настройте обрезку — перетаскивайте и масштабируйте изображение в редакторе
3. Выберите режим (**API** или **Рисование**) и настройки качества
4. Нажмите **«Сгенерировать скрипт»**
5. Скопируйте скрипт и вставьте в консоль браузера (`F12` → Console) на сайте **ИТД**

## 📱 Мобильная версия

Мобильный инжектор (`m.js`) добавляет интерфейс загрузки баннера прямо на сайт ИТД.

### Скрипт для вставки

```javascript
fetch('https://nersuga.github.io/itd_imager/m.js').then(r=>r.text()).then(t=>{const s=document.createElement('script');s.textContent=t;document.head.appendChild(s)})
```

### 🤖 Android

1. Установите [**Via Browser**](https://play.google.com/store/apps/details?id=mark.via) или [**Kiwi Browser**](https://play.google.com/store/apps/details?id=com.kiwibrowser.browser)
2. Откройте `итд.com` и войдите в аккаунт
3. Откройте консоль:
   - **Via:** `⋮` → Инструменты → **JavaScript**
   - **Kiwi:** `⋮` → **Консоль разработчика**
4. Вставьте скрипт и нажмите **Выполнить** / **Enter**
5. На сайте появится кнопка 🎨 — нажмите и загрузите фото!

### 🍎 iPhone / iPad

1. Откройте **Safari** → `итд.com` → войдите в аккаунт
2. Создайте **любую закладку** (📤 → «Добавить закладку»)
3. Откройте **Закладки** → найдите созданную → нажмите **Править**
4. Замените URL закладки на скрипт, добавив `javascript:` в начало
5. Перейдите на `итд.com` → нажмите на эту закладку из адресной строки

> 💡 **Альтернатива:** Установите [Web Inspector](https://apps.apple.com/app/a-web-inspector/id1584825745) из App Store → подключите к Safari → откройте консоль → вставьте скрипт.

### Возможности мобильной версии

- Выбор типа: Баннер (1100×380) / Пост (800×500)
- Зум двумя пальцами (pinch-to-zoom) и перетаскивание
- Автоматическая авторизация и загрузка на ИТД
- Прогресс-бар с обратной связью
