## ✨ Возможности

| Фича | Описание |
|-------|----------|
| 🖱️ **Drag-crop редактор** | Перетаскивание, zoom (колёсико / pinch), rule of thirds |
| ⚡ **API-режим** | Загрузка оригинального изображения через API сайта |
| 🎨 **Режим рисования** | Квантизация палитры (2–256 цветов) + загрузка |
| 📱 **Мобильный букмарклет** | Работает без консоли DevTools |
| 🔧 **Выбор качества** | PNG, JPEG (95/85/70%), WebP (90/75%) |
| 📐 **Автодетект устройства** | Размер и формат подбираются по экрану |

## 🚀 Быстрый старт

Проект — чистый HTML/CSS/JS без зависимостей. Никакой сборки не нужно.

### Вариант 1: Открыть напрямую

```
Откройте index.html в браузере
```

### Вариант 2: Локальный сервер

```bash
# Python
python -m http.server 8080

# Node.js (npx)
npx -y serve .

# VS Code
# Используйте расширение Live Server
```

Затем откройте `http://localhost:8080`

## 📖 Как использовать

### Десктоп (API-режим)

1. Загрузите изображение в ITD Imager
2. Настройте обрезку (drag/zoom) и качество
3. Нажмите **«Сгенерировать скрипт»**
4. Откройте [итд.com](https://итд.com) → `F12` → Console → вставьте скрипт

### Мобильные (букмарклет)

1. На странице ITD Imager нажмите **«Установить букмарклет»** → сохраните в закладки
2. Откройте [итд.com](https://итд.com) и залогиньтесь
3. Запустите закладку → выберите фото → готово!

## 🔧 Технологии

- **Vanilla JS** — без фреймворков и зависимостей
- **Canvas 2D API** — crop editor, превью, квантизация
- **OffscreenCanvas** — GPU-ускоренная обработка
- **Median-Cut + HashMap** — быстрая квантизация цветов
- **Pointer Events** — поддержка мыши и тач-устройств
