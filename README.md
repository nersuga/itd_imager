# 🎨 ITD Imager

> Веб-инструмент для подготовки и автозагрузки изображений (баннеров/постов) на платформу **ИТД**.

![HTML](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

## ✨ Возможности

- **Загрузка изображений** — drag & drop или выбор файла (PNG, JPG, WebP, GIF)
- **Обрезка и масштабирование** — интерактивный редактор с зумом, панорамированием и сеткой третей
- **Два режима работы:**
  - ⚡ **API** — генерирует скрипт для прямой загрузки через API платформы
  - 🎨 **Рисование** — квантизация цветов (Median-Cut) с предпросмотром палитры
- **Настройки** — выбор размера холста (Баннер 1100×380 / Пост 800×500), формата (PNG, JPEG, WebP) и качества
- **Мобильная поддержка** — букмарклет для загрузки прямо с телефона
- **Адаптивный дизайн** — тёмная тема, современный UI

## 📦 Установка

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/<ваш-username>/itd_imager.git
   cd itd_imager
   ```

2. **Зависимости не требуются** — проект работает на чистом HTML/CSS/JS без сборщиков.

## 🚀 Запуск

Откройте файл `index.html` в браузере — или запустите через любой локальный сервер:

```bash
# Python
python -m http.server 8080

# Node.js (npx)
npx serve .

# VS Code — расширение Live Server
```

Затем перейдите по адресу `http://localhost:8080`.

## 🛠 Как пользоваться

1. Загрузите изображение (перетащите или нажмите на область загрузки)
2. Настройте обрезку — перетаскивайте и масштабируйте изображение в редакторе
3. Выберите режим (**API** или **Рисование**) и настройки качества
4. Нажмите **«Сгенерировать скрипт»**
5. Скопируйте скрипт и вставьте в консоль браузера (`F12` → Console) на сайте **ИТД**
