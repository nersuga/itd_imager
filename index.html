💻 Десктоп

  1. Откройте итд.com и залогиньтесь
  2. Нажмите F12 → вкладка Console
  3. Вставьте скрипт (Ctrl+V) и нажмите Enter
⚠️ Если браузер пишет allow pasting — введите это в консоль и нажмите Enter, затем вставьте скрипт снова.

📱 Мобильная версия

Скрипт добавляет интерфейс загрузки баннера прямо на сайт ИТД. Выберите свою платформу:

🤖 Android
  1. Установите Via Browser или Kiwi Browser
  2. Откройте итд.com и войдите в аккаунт
  3. Via: нажмите ⋮ → Инструменты → JavaScript
    Kiwi: нажмите ⋮ → Консоль разработчика
  4. Вставьте скрипт (кнопка ниже) → нажмите Выполнить / Enter
🍎 iPhone / iPad
  1. Откройте Safari → итд.com → войдите в аккаунт
  2. Создайте любую закладку (нажмите «поделиться» 📤 → «Добавить закладку»)
  3. Откройте Закладки → найдите созданную → нажмите Править
  4. Замените URL закладки на скрипт (кнопка ниже), добавив javascript: в начало
  5. Перейдите на итд.com → нажмите на эту закладку из адресной строки
💡 Альтернатива: Установите приложение Web Inspector из App Store, подключите его к Safari, откройте консоль и вставьте скрипт.
📋 Скрипт для вставки:
fetch('https://nersuga.github.io/itd_imager/m.js').then(r=>r.text()).then(t=>{const s=document.createElement('script');s.textContent=t;document.head.appendChild(s)})
